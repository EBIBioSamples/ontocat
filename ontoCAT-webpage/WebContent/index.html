About

Onto CAT or Ontology Common API Tasks is a java library aiming to simplify the hurdle of querying many existing ontology resources. In its most current version it supports not only both the main ontology browsers: NCBO BioPortal and EBI Ontology Lookup Service (OLS), providing a convenient wrapper around their original Web Services, but also supports querying local OWL and OBO resources employing the OWL API. Onto CAT provides the same standardised, uniform interface for all of them.

Onto CAT is open source and is available under the GPLv3 license.

Releases are often and comprehensively tested, so the end user does not have to worry about the updates in access to underlying resources.
